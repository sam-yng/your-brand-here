# Sunday Service and Common Thread

Status: queued after Northline architecture proof.

## Objective

Complete the range proof with two visibly distinct fictional systems while preserving
shared accessibility, content honesty, progressive enhancement, and bundle limits.

## Dependencies

- Northline exit confirms study scoping pattern.
- Study font subset of EXT-09 available, or temporary fallbacks explicitly tracked.

## Scope and order

1. Build Sunday Service namespace, hero, truthful fictional-release language, shift
   content, feature cards, build note, and CTA.
2. Implement shift controls with all content server-rendered and programmatic selected
   state. Keep wishlist as a non-successful concept unless a real production boundary
   is defined; never reproduce prototype-only optimistic success.
3. Build Common Thread namespace, hero, six-product lookbook, URL-backed category
   filters, sizing/care content, build note, and CTA.
4. Implement filters as ordinary links first, then enhance without losing URL state.
5. Implement sizing/care trigger, region/drawer semantics, predictable focus, close
   behavior, and complete print CSS.
6. Add route-owned CreativeWork metadata inputs and focused tests without sharing
   visual layout components across studies.

## External work

  under EXT-10.
- No wishlist provider is required for v1; if scope changes, open a separate decision

## Verification

- Desktop/mobile comparisons to both accepted prototypes.
- JavaScript-disabled page contains every Sunday shift and usable Common Thread
  filter links/sizing content.
- Keyboard, focus, selected-state, reduced-motion, and drawer recovery checks pass.
- Common Thread print output retains products, sizing, care, and contact context.
- Study CSS/fonts do not leak and pages remain visibly different systems.
- Each study compressed JavaScript stays at or below 90KB.
- `bun run check` passes.

## Exit

All three studies are complete, honestly labelled, individually authored, and pass
the shared accessibility/performance floor.
